# GTA / FiveM Mod Manager

برنامج إدارة مودات للاعبين — جرافكس، أصوات أسلحة، وشكل/سكن أسلحة.

## التشغيل

```bash
npm install
npm run dev
```

أو دبل كليك على `run.bat`

## إضافة المودات (سهل!)

افتح مجلد `mods/` أو اضغط **📁 المودات** من البرنامج:

```
mods/
├── graphics/          ← جرافكس (ملفات citizen)
│   └── اسم-المود/
│       └── citizen/common/data/timecycle_mods_1.xml
│
├── audio/             ← أصوات أسلحة (GTA)
│   └── اسم-المود/
│       └── WEAPONS_PLAYER.rpf
│
└── weapons/           ← أسلحة (FiveM)
    └── اسم-المود/
        └── w_ar_assaultrifle.ytd
```

### الخطوات:
1. أنشئ مجلد باسم المود داخل التصنيف المناسب
2. ضع ملفات المود داخله (مباشرة أو بمجلد citizen/)
3. اضغط **🔄 تحديث** في البرنامج
4. المود يظهر تلقائياً!

### تخصيص الاسم (اختياري):
أنشئ `info.json` داخل مجلد المود:
```json
{
  "nameAr": "سماء جميلة",
  "descriptionAr": "سماء واقعية مع غروب رائع"
}
```

## أين تروح الملفات؟

| التصنيف | المسار في اللعبة |
|---------|-----------------|
| graphics | `FiveM/citizen/common/data/` |
| audio | `GTA/x64/audio/sfx/` |
| weapons | `FiveM/mods/` |

1. اضغط ⚙️ وحدد مسارات GTA V و FiveM
